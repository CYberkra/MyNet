## 版本化根目录文档归档

自 v0.8.0-alpha.29 起，`docs/` 根目录只保留稳定文件名；带 `v080aXX`、旧 UI 版本号或阶段性修复说明的文档归档到：

- `docs/history/versioned_root_docs/`
- `docs/history/real_data/`
- `docs/history/ui_overview_images/`

这样后续软件迭代时，不再让当前文档入口保留过期版本号。
