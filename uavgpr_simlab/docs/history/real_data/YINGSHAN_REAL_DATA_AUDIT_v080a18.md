## 需要继续人工确认的合同

1. 钻孔柱状图 PDF 与 ZK07-ZK10 的一一对应关系，需要在后续人工/半自动解析后固化成 `configs/boreholes_yingshan.csv`。
2. 当前附件中显式存在 3、6、7、9 号测线剖面 PDF；L1、X1 的剖面来源需要确认是否在 DWG 或其他图件中。
3. 真实数据进入 GUI 前，应增加一个项目级 `real_data/` 导入向导，而不是把绝对路径写死到源码或默认工作区。
4. 基覆界面误差验证需要把成像界面深度与钻孔揭示深度纳入同一坐标/高程基准后再计算，不能只凭截图判断。
