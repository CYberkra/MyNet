# UavGPR-SimLab v0.7 产品化易用界面重构报告

## 目标

v0.7 不是继续增加底层仿真功能，而是把 v0.6 已经具备的功能做成更接近“普通用户一眼看懂”的产品界面：

- 少术语，多图像；
- 首页直接看到当前项目状态；
- 模型预览像图库一样选择模型；
- 批量仿真像任务看板一样显示进度；
- 历史记录像档案卡一样同时显示模型图和 B-scan 图；
- 高级参数继续保留，但默认不打扰普通用户。

## 本版已完成的真实 GUI 改动

### 1. 整体视觉系统

- 重新设计了全局 QSS 样式；
- 增强圆角卡片、状态标签、流程节点和分区标题；
- 侧边栏改为更明确的 6 个入口：
  - 首页
  - 模型预览
  - 批量仿真
  - 历史与结果
  - 项目管理
  - 设置与帮助

### 2. 首页产品化

首页现在更接近项目状态大屏：

- 当前项目
- 已生成模型数
- 正在仿真数
- 已完成数
- 需检查数
- 最近 B-scan 预览
- 下一步操作建议
- “生成模型 → 看模型 → 批量仿真 → 看 B-scan → 历史复盘”的流程卡片

### 3. 模型预览图库化

模型预览页从工程式列表升级为“模型图库”：

- 左侧模型缩略图列表；
- 中间大模型预览；
- 右侧用普通话解释：
  - 地表起伏
  - 覆盖层厚度
  - 是否有障碍物
  - 模型难度
  - 推荐用途

### 4. 批量仿真任务看板化

批量仿真页强化了：

- 顶部任务统计卡片；
- 自动跳过重复任务提示；
- 生成模型 → 检查 → 仿真 → 后处理 → 查看结果流程；
- 每个任务保留缩略图、状态、进度、完成道数、仿真内容、操作提示。

### 5. 历史与结果档案卡化

历史页现在更接近“结果档案馆”：

- 每条历史记录显示：
  - 模型缩略图；
  - B-scan 缩略图；
  - 状态；
  - 完成道数；
  - 时间；
- 右侧保留模型大图和 B-scan 大图；
- 继续支持刷新、导出、重跑、删除本次记录。

## 没有改动的稳定底层

v0.7 没有重写以下稳定逻辑：

- gprMax 调用；
- 任务去重；
- history marker；
- 运行中 B-scan 读取；
- 完成后 B-scan 读取；
- manifest 与 labels.json 生成；
- 后处理导出。

因此 v0.7 主要是 GUI 表现层升级，底层仍沿用 v0.5.5/v0.6 已审计过的稳定链路。
