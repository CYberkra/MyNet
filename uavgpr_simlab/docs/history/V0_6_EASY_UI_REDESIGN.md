# UavGPR-SimLab v0.6 易用化界面重构说明

## 目标

v0.6 将默认 GUI 从“工程参数页签式界面”改成“用户一眼看懂的工作台”。目标用户可以不理解 gprMax、variant、manifest、SFCW 等名词，也能直观看到：

- 自己生成的模型长什么样；
- 这一批做了哪些模型；
- 仿真现在跑到哪里；
- 已完成和运行中的 B-scan 是什么样；
- 历史记录如何导出、删除、重跑。

## 默认入口

```bash
PYTHONPATH=src python -m uavgpr_simlab.app
```

默认打开 v0.6 易用版界面。

旧工程界面仍保留：

```bash
PYTHONPATH=src python -m uavgpr_simlab.app --advanced
```

## 新界面结构

左侧导航只保留 6 个普通用户能理解的页面：

1. 首页
2. 模型预览
3. 批量仿真
4. 历史与结果
5. 项目管理
6. 设置与帮助

## 核心改动

### 1. 首页

首页改成大卡片仪表盘，显示当前项目、已生成模型数、正在仿真数、已完成数、需检查数，以及最近 B-scan 预览。

### 2. 模型预览

模型预览页改成“左侧缩略图列表 + 中间 3D/2.5D 模型大图 + 右侧普通话说明”。模型信息不再优先显示复杂配置，而是显示：

- 地表起伏；
- 覆盖层厚度；
- 是否有障碍物；
- 模型难度；
- 推荐用途。

### 3. 批量仿真

批量页改成任务表，每行一个任务，并显示：

- 模型缩略图；
- 模型名称；
- 状态；
- 进度；
- 已完成道数；
- 标签/variant；
- 操作提示。

默认仍使用 v0.5.5 的去重机制，不会重复跑相同指纹的任务。

### 4. 历史与结果

历史页改成图像复盘页面：

- 左侧每次仿真都有模型缩略图；
- 右侧显示模型大图和 B-scan 大图；
- 运行中的任务可显示已产出的 trace 合成 B-scan；
- 支持导出历史 CSV、重跑、删除记录。

### 5. 项目管理

把工作目录、仿真计划和模型数量集中到一个页面，减少新用户在多个页签之间找参数的负担。

### 6. 设置与帮助

只保留必要环境设置：gprMax 源码目录、conda 环境名、GPU ID、OpenMP 线程。高级工程界面作为按钮入口保留。

## 技术说明

- 新增 `src/uavgpr_simlab/gui/easy_window.py`；
- 修改 `src/uavgpr_simlab/app.py`，默认启动易用版；
- 旧版 `main_window.py` 未删除，可通过 `--advanced` 或界面按钮打开；
- 批量运行、去重、历史记录、B-scan 读取仍复用 v0.5.5 的核心模块，避免另起一套逻辑导致不一致。
