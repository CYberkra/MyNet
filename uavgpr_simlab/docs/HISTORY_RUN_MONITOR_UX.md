# 历史与运行监控体验说明

版本：v0.8.0-alpha.35

本轮围绕“导入数据集骨架后一键运行，并能看到已跑、正在跑、即将跑的模型”继续优化：

1. 批量页保留运行中 B-scan 大图，同时新增最近完成 B-scan 对比条。
2. 最近对比条最多显示 5 个完成的 case/variant，便于连续观察 raw、target_only、clutter_only、air_only 等输出是否正常。
3. 历史页支持在数据集 / case / variant 树中右键操作。
4. 右键菜单支持打开 case 文件夹、打开 QC JSON、复制失败原因、复制路径摘要、切换到批量页只重跑 failed。
5. pending 虚拟记录只用于查看和预检，不允许直接当作历史 marker 删除。

后续可继续把“右键重跑当前 case / 当前 variant”“运行中缩略图固定 pin 住”和“失败原因聚合面板”做成独立任务。
